# 描述

此文件用来说明, 音乐 JSON 文件的数据格式.

即: 说明如何描述一个音乐的歌词, 起始时间, 以及在游戏中, 怎么按下游戏琴键的操作

```js
/** 这描述整首歌该如何操作
 * 每个子对象表示: 这行歌词该如何在游戏中操作
 */
[
  // 子对象
  {
    // 显示歌词
    words: "我要这天再遮不住我眼",

    /** 歌词的起始时间, 单位秒 (s)
     * 即: 当前歌词, 应该 time 秒时播放
     */
    time: 10,

    /** 在游戏, 歌词如何进行按键操作
     * 每个子数组表示: 按下一次
     * 为什么使用子数组, 而不是子对象?
     * 因为考虑到需要有混音, 即: 同一时间, 多个键一起按下的情况
     */
    data: [
      /** 按下第 1 次
       * 每个子对象表示: 按下的键
       */
      [
        // 这个子数组, 延迟 500ms, 才会按下它描述的键
        500,
        {
          key: 1, // 500ms 后, 又延迟 10ms, 按下键 1
          delay: 10, // 此键延迟多久按下  ms
          pressDuration: 2000, // 此键按下持续时间(ms), 默认为 1000ms
        },
        {
          // 由于键 1 延迟 10ms, 所以会先按下 2, 然后过 10ms 才按下 1
          key: 2, // 和键 1 一起等待 500ms 后,  按下键 2
        },
      ],

      /** 按下第 2 次
       *  这里将同时按下 3 和 4 游戏键
       */ 
      [
        0, // 不延迟, 第 1 次按键结束, 直接按下 3, 4
        {
          key: 3,
        },
        {
          key: 4,
        },
      ],
    ],
  },
];
```
